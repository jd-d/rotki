# Dev Goals
- Advanced VPN routing (priority) – support multiple `.ovpn` configs, per-endpoint routing, round-robin/parallel tunnels for improved privacy.
- Remove artificial UI limitations – ensure the local build exposes all features without remote toggles.
- Network telemetry view – log outbound/inbound calls with their routing, and surface them in a dedicated UI section (potentially alongside VPN config).
- Operational workflow – keep Codex + WSL build/test loops aligned while avoiding merge drift.
